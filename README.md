Sistema de Gestión con Login, Roles y Bodega (Java + ORM/JPA + MySQL)

Este proyecto implementa un sistema académico desarrollado en Java con persistencia mediante ORM/JPA. Incluye autenticación, gestión de usuarios, administración de productos en bodega y lógica de negocio con validaciones reales.

* Funcionalidades principales

  Autenticación con roles (Administrador y Usuario)
  Usuario administrador precargado: admin / admin
  CRUD de Usuarios (solo Administrador)
  Gestión de Bodega
    Registro de productos
    Ingreso y salida de cajas
    Control de stock
    Validación de edad mínima
  Módulo de compras (solo Usuario)

* Tecnologías utilizadas

  Java
  ORM / JPA
  MySQL
  Programación Orientada a Objetos

* Características técnicas

  Cifrado de contraseñas en MD5
  Validación de códigos únicos
  Restricción de stock
  Validación de edad mínima
  Estructura modular por roles
